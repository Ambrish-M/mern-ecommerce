

### 🛒 E-Commerce App (MERN + Razorpay + Zustand)
A full-featured E-commerce web application built using the **MERN stack**. It supports user authentication, product creation, cart management, order placement, and real-time Razorpay payment integration. State management is handled using **Zustand**, and all HTTP interactions are managed via **Axios**.

**Tech Stack:**  
`MongoDB` • `Express` • `React` • `Node.js` • `Zustand` • `Razorpay` • `Axios`  
**Features:** 🛒 💳 🔐 📦 📱  
- Product Management  
- Cart & Checkout System  
- Razorpay Payment Integration  
- Admin Dashboard  
- Fully Responsive Design
